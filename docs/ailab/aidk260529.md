## 1-YOLOV5
<br>
Run **1-YOLOV5** with the following steps:

1. **Open main.ipynb**

    Click directory  **01-yolov5** → double click file **main.ipynb**

2. **Let infer_mode = 'image'**

    Change code of **main.ipynb**, let `infer_mode = 'image'` 

3. **Click double arrow ▶︎▶︎ on top of browser，then click button <ins>Restart</ins>**

    ![aidk-01yolors](https://tnt.gdvzz.com/aikit/aidk.assets/aidk-01yolors.jpg)
    
    Get infer result few seconds later:

    ![aidk-01yoloi](https://tnt.gdvzz.com/aikit/aidk.assets/aidk-01yoloi.jpg)

4. **Let infer_mode = 'video'**

5. **Click double arrow ▶︎▶︎ on top of browser，then click button <ins>Restart</ins>**

    Get infer result few seconds later：

    ![aidk-01yolov](https://tnt.gdvzz.com/aikit/aidk.assets/aidk-01yolov.jpg)

6. **Let infer_mode = 'camera'**

7. **Click double arrow ▶︎▶︎ on top of browser，then click button <ins>Restart</ins>**
